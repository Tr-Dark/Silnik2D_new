Detailed Description

Klasa zarządzająca ładowaniem i przechowywaniem bitmap.

Klasa umożliwia ładowanie tekstur z plików oraz dostęp do nich poprzez indeksy. Używana jest w aplikacjach graficznych korzystających z SFML.

◆ loadFromFile()

bool BitmapHandler::loadFromFile ( const std::string & filename,
int bitmapIndex )

Ładuje teksturę z pliku i zapisuje ją w podanym indeksie.

Ładuje teksturę z pliku do wektora tekstur.

Funkcja próbuje załadować plik graficzny do tekstury. Jeśli operacja się powiedzie, tekstura zostaje przechowana w wewnętrznym wektorze pod wskazanym indeksem.

Parameters
filenameŚcieżka do pliku graficznego.
bitmapIndexIndeks, pod którym tekstura ma zostać zapisana.
Returns
true Jeśli tekstura została pomyślnie załadowana.
false Jeśli nie udało się załadować tekstury.

Ta metoda ładuje plik tekstury z podanej ścieżki do wektora tekstur w odpowiednim miejscu na podstawie indeksu bitmapy. Jeśli indeks bitmapy jest większy niż rozmiar wektora, rozmiar wektora zostaje zwiększony.

Parameters
filenameŚcieżka do pliku z teksturą.
bitmapIndexIndeks bitmapy, gdzie tekstura ma zostać załadowana.
Returns
true, jeśli tekstura została załadowana pomyślnie; false w przeciwnym razie.
