Detailed Description

Abstrakcyjna klasa bazowa dla obiektów animowanych.

Ta klasa definiuje interfejs do obiektów, które posiadają animację. Klasa jest przeznaczona do dziedziczenia przez inne klasy, które implementują metodę animacji w zależności od swojego typu.
