Detailed Description

Główna klasa silnika gry.

Klasa odpowiedzialna za zarządzanie główną pętlą gry, renderowaniem, obsługą zdarzeń oraz logiką gry. Umożliwia rysowanie obiektów graficznych, obsługę gracza oraz zarządzanie tłem.
